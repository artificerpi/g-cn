# G-CN
g-cn 服务白名单列表 (尝试原生安卓体验的临时解决方案)

## Usage
``` 
Usage of gcn:
  -dst string
        destination file path (default "glist.txt")
  -encode
        specify whether it's encoding, set false for decoding (default true)
  -src string
        source file path (default "g-cn.txt")
```
